# SVG to PNG Converter

一个基于 Vue 3 + Vite 的纯前端 SVG 转 PNG 工具。

## ✨ 特性

-   **纯前端处理**: 所有转换都在浏览器本地完成，保护隐私。
-   **实时预览**: 拖拽即预览，支持透明背景查看。
-   **灵活设置**: 自定义输出尺寸、缩放比例，支持锁定宽高比。
-   **现代 UI**: 采用玻璃拟态设计风格，界面美观流畅。

## 🛠️ 技术栈

-   Vue 3 (Composition API)
-   Vite
-   TypeScript
-   Tailwind CSS

## 🚀 快速开始

1.  安装依赖:
    ```bash
    npm install
    ```

2.  启动开发服务器:
    ```bash
    npm run dev
    ```

3.  构建生产版本:
    ```bash
    npm run build
    ```
